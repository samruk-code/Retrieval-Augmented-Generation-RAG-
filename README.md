# Retrieval-Augmented Generation (RAG)

Completed coursework from the **[Retrieval Augmented Generation (RAG)](https://www.deeplearning.ai/courses/retrieval-augmented-generation-rag/)** course by [DeepLearning.AI](https://www.deeplearning.ai/), taught by **Zain Hasan** (Senior AI/ML Developer Relations Engineer, Together.ai).

> Build production-ready RAG applications — from architecture and retrieval techniques to vector databases, evaluation, and deployment.

## Modules & Labs

### Week 1 — RAG Fundamentals
Core concepts of RAG, large language model basics, and information retrieval foundations.

| Lab | Type | Description |
|---|---|---|
| `W1/Lab1` | Ungraded | Intro to RAG pipeline components |
| `W1/Lab2` | Ungraded | LLM integration and basic retrieval |
| `W1/Graded Lab 1` | **Graded** | End-to-end RAG pipeline on news data |

---

### Week 2 — Search Techniques
Keyword search (TF-IDF, BM25), semantic search, hybrid retrieval, and retrieval evaluation metrics.

| Lab | Type | Description |
|---|---|---|
| `W2/Lab1` | Ungraded | Semantic search with embeddings |
| `W2/Lab2` | Ungraded | Hybrid search and evaluation |
| `W2/Graded Lab 2` | **Graded** | Implementing and evaluating search strategies |

---

### Week 3 — Vector Databases & Advanced Retrieval
Working with Weaviate, document chunking strategies, query parsing, and reranking.

| Lab | Type | Description |
|---|---|---|
| `W3/Lab1` | Ungraded | Weaviate API and vector storage |
| `W3/Lab2` | Ungraded | Chunking and query parsing |
| `W3/Graded Lab` | **Graded** | Full retrieval pipeline with Weaviate + Flask |

---

### Week 4 — Generation & Prompt Engineering
Transformer architecture, prompt engineering, hallucination handling, and agentic RAG patterns.

| Lab | Type | Description |
|---|---|---|
| `W4/Lab1` | Ungraded | Prompt engineering for RAG |
| `W4/Lab2` | Ungraded | Agentic RAG and tool use |
| `W4/Graded Lab` | **Graded** | Fine-tuning format + agentic pipeline with Weaviate |

---

### Week 5 — Production, Evaluation & Monitoring
Deployment strategies, evaluation frameworks, observability, and multimodal RAG.

| Lab | Type | Description |
|---|---|---|
| `W5/Lab1` | Ungraded | Monitoring and observability |
| `W5/GradedLab` | **Graded** | Production-ready RAG system with evaluation |

## Skills Demonstrated

- Designing and implementing end-to-end RAG pipelines
- Semantic search, keyword search (TF-IDF / BM25), and hybrid retrieval
- Vector database integration with **Weaviate**
- Document chunking, embedding generation, and reranking
- Prompt engineering and hallucination mitigation
- Agentic RAG patterns and tool use
- RAG evaluation metrics and monitoring
- Serving RAG systems via **Flask**

## Tech Stack

`Python` · `Jupyter Notebooks` · `Weaviate` · `Flask` · `Sentence Transformers` · `scikit-learn`
